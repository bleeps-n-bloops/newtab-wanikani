SETTINGS

Heres the place where you need to provide your WaniKani key in order for us to be able to display information to you! Not sure where to get your key? Simply head to the settings page on WaniKani (which can be found here) and grab a key for the API v1! From there enter it in the input box below and hit the save button, we will automagically do the rest and give you the information everytime you make a new tab!
